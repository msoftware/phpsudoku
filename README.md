Install:
	- Copy all files to the webspace
	- Unzip sudoku.txt.gz to sudoku.txt (You can use 7Zip http://www.7-zip.de/ to unzip the file)
	- Copy sudoku.txt to the webspace
	- Ready to play sudoku
